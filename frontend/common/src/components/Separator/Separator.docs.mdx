# Separator

> Visually or semantically separates content.

This is a styled version of [@radix-ui/separator](https://www.radix-ui.com/docs/primitives/components/separator).

## Usage

There are two important things to keep in mind when using a separator:

 - `orientation` - Horizontal or Vertical
 - `decorative` - A meaningful separator or decorative
   - See: [separator role](https://www.w3.org/TR/wai-aria-1.2/#separator:~:text=separator%20role-,A%20divider%20that%20separates%20and%20distinguishes%20sections%20of%20content%20or%20groups%20of%20menuitems.,-There%20are%20two)

<Story id="components-separator--default" />

```tsx
import Separator from "@common/components/Separator";

const SeparatorExample = () => (
  <Separator
    decorative
    orientation="vertical"
    color="secondary"
  />
);
```
## Props

All props from the `@radix-ui/separator` as well as additional style props.

See: [Documentation](https://www.radix-ui.com/docs/primitives/components/accordion#trigger)

### Definition

```tsx
export interface SeparatorProps extends SeparatorPrimitiveRootProps {
  color?: Color;
}
```

 - `color`: Hydrogen color to display the separator as
