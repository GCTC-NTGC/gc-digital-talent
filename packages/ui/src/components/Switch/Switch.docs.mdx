# Switch

> A control that allows the user to toggle between checked and not checked.

This is a styled version of [@radix-ui/switch](https://www.radix-ui.com/docs/primitives/components/switch).

## Subcomponents

- `Switch.Root`
- `Switch.Thumb`

## Usage

<Story id="components-switch--default" />

```tsx
import Switch from "@gc-digital-talent/ui/Switch";

const SwitchExample = () => (
  <Switch.Root id="example-switch">
    <Switch.Thumb />
  </Switch.Root>
);
```
